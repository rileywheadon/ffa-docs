Stationary FFA

Flood frequency analysis (FFA) is a technique used to make data-driven estimates of the severity of streamflow events. We often express the results of FFA in terms of return periods and return levels. The return period of a flood is the expected number of years between streamflow events of equal or greater severity. The return level is the magnitude of the streamflow event, measured in \(\text{m}^3/\text{s}\).

In this vignette, we will perform FFA on an example dataset using the ffaframework package. We will assume that the time series is stationary, which means that the underlying probability distribution is constant in time.

Setup

We begin by loading the example dataset Application_1.csv, which has two columns:

  • max: the AMS measurements
  • year: the corresponding observation year
library(ffaframework)
csv_path <- system.file("extdata", "Application_1.csv", package = "ffaframework")
df <- read.csv(csv_path)
df <- subset(df, !is.na(max))

Distribution Selection

In the first stage of FFA, we select a suitable probability distribution for the data using the method of L-moments (Hosking, 1997). The ffaframework package provides three selection metrics involving the L-moments:

  1. select_ldistance compares the sample L-skewness (\(t_3\)) and sample L-kurtosis (\(t_4\)) with the theoretical L-skewness and L-kurtosis of each distribution. The distribution whose theoretical L-moments have the lowest Euclidian distance from the \((t_3, t_4)\) point is selected.
  2. select_lkurtosis compares the sample L-kurtosis with the theoretical L-kurtosis of each probability distribution. The distribution whose theoretical L-kurtosis is the closest to the sample L-kurtosis is selected.
  3. select_zstatistic fits a 4-parameter Kappa distribution to the data using the method of L-moments. Then, samples are drawn from this distribution to empirically estimate the distribution of the theoretical L-kurtosis. Next, a z-statistic is computed for each distribution using the bootstrap distribution and the sample L-moments. The distribution with the lowest Z-statistic is selected.

We will use the select_ldistance metric in this vignette.

The plot_lmom_diagram function displays the results of distribution selection.

selection <- select_ldistance(df$max)

print(selection$recommendation)
#> [1] "GEV"

plot_lmom_diagram(selection)

We can see that select_ldistance recommended the generalized extreme value (GEV) distribution. For information about the other distributions, see the selection$metrics item. On the L-moments diagram, we can see that the GEV distribution (indicated by the yellow line), is the closest to the sample L-moments (indicated by the black triangle). You can find more information about the probability distributions supported by the framework here.

Parameter Estimation

Now that we have chosen a distribution, we need to fit the distribution to the data by estimating the parameters. The ffaframework package provides three methods for parameter estimation: L-moments, maximum likelihood, and generalized maximum likelihood. See here for more information.

In this vignette, we will use L-moments parameter estimation. Each distribution has its own method for L-moment parameter estimation with the prefix fit_lmom. Since distribution selection recommended the generalized extreme value (GEV) distribution, we will use the fit_lmom_gev function to estimate the parameters.

params <- fit_lmom_gev(df$max)

print(params)
#> [1] 1600.219872  616.666030    0.120747

Uncertainty Quantification

Now that we have a fitted probability distribution, we can easily estimate the return levels. However, reporting a single point estimate for each return period is not best practice. Instead, we strongly advise performing uncertainty quantification to determine the confidence intervals of each return level.

The uncertainty_bootstrap function performs uncertainty quantification using the sample bootstrap method. It requires three arguments:

  • data: A vector of annual maximum streamflow observations.
  • model: A three-letter code for a probability distribution (ex. "GEV")
  • method: A parameter estimation method. Must be "L-moments", "MLE", or "GMLE".

By default, the uncertainty_bootstrap method computes return levels and confidence intervals for the 2, 5, 10, 20, 50, and 100 year return periods. The plot_uncertainty method generates a plot of the results.

uncertainty <- uncertainty_bootstrap(df$max, "GEV", "L-moments")

plot_uncertainty(uncertainty)

Model Assessment

The ffaframework package also provides the model_diagnostics function, which reports a collection of assessment statistics about the flood frequency analysis. Many of these assessment statistics are determined by comparing the predictions of the parametric model with the empirical plotting positions (see here for more information).

The plot_model_diagnostics generates a visual comparison of the empirical plotting positions (the “Observed Quantiles”) and the predictions of the parametric model (the “Model Quantiles”). The black line represents a perfect 1:1 correspondence between the model and the data.

diagnostics <- model_diagnostics(df$max, "GEV", params, uncertainty)

plot_model_diagnostics(diagnostics)

From the plot above, we can see that the parametric model is mostly aligned with the empirical plotting positions, with the exception of a small negative bias between \(3000\text{m}^3/\text{s}\) and \(4500\text{m}^3/\text{s}\).
