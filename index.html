Welcome to the FFA Framework wiki!

The FFA Framework is an open-source project for Flood Frequency Analysis (FFA) developed by researchers at the University of Calgary, Canada1. The framework is currently available as an R package and a command line interface (CLI).

  • To get started with the R package, please read the documentation.
  • To get started with the CLI, please read the installation instructions and the user manual.

You may also be interested in our goals, roadmap, or release notes.

Warning: This version of the FFA Framework is incomplete.
